How it works

Three steps. Ten seconds.

No plugins, no API keys, no admin access. Paste, pick, download.

01

Paste a URL

Drop in the address of any public WooCommerce or Shopify store. We read its public product API — no plugins, no API keys, no access to the admin.

02

Choose your format

Export as a WooCommerce product CSV or a Shopify product-import CSV. Each uses that platform's exact column layout, so it drops straight into the native importer.

03

Download the CSV

Get the file instantly. Variations, images, SKUs, prices and stock levels are all included and ready to import — nothing to clean up by hand.
